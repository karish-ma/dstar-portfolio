# Digital STARs - Curriculum Architecture Portfolio

An interactive single-page portfolio showcasing the curriculum matrix for **Digital STARs** (Digital Skills Training for All Romanians), a World Bank-supported national digital literacy programme under Romania's National Recovery and Resilience Plan (NRRP).

## Live Site

**[karish-ma.github.io/dstar-portfolio](https://karish-ma.github.io/dstar-portfolio/)**

## About the Programme

Digital STARs aims to train 100,000 Romanian citizens in foundational digital skills by June 2026, delivered through a national network of 700+ public libraries. The programme is led by Romania's Ministry of Research, Innovation and Digitalization (MoRID) with technical assistance from the World Bank and implemented by the Altamont Group.

## Curriculum Architecture

- **6 Learning Units** mapped to DigCompRo (Romania's national Digital Competence framework, aligned to EU DigComp)
- **3 Life Tracks** - Private (5 hrs), Professional (6 hrs), and Citizen (2 hrs, mandatory for all)
- **4-Stage Assessment** - EU Digital Skills Indicator pre-test, embedded self-reflection, DSI post-test, and randomised certification bank
- **Blended delivery** - self-paced eLearning on Moodle (SCORM 1.2) + librarian-facilitated in-person sessions
- **EN 301 549** accessibility compliance

## Portfolio Features

- Filter learning units by DigCompRo competency area
- Filter by learner track (Private / Professional / Citizen)
- Expand each unit for full instructional detail: objectives, key messages, strategies, and assessment
- Visual fork/merge pathway diagram showing how the three tracks connect
- KSA (Knowledge, Skills, Attitudes) framework breakdown with proficiency levels

## Context

Built as a portfolio piece for curriculum design roles, demonstrating backward design methodology, standards-aligned instructional architecture, and large-scale blended learning programme design.
